All Fairphones after 2 require you to request a code from [this][unlock-website] website to enable OEM unlock.The process for enabling bootloader unlocking is changing for Fairphone 3. [Since version 6.A.040.2 (March 2026)], you do not need to enter a code anymore. Just follow these steps:
1
Go to the Settings app on the phone in the section "About Phone"

2
Tap on the Build number seven times to enable Developer Options

3
Toggle Bootloader Unlocking in Developer options

4
Wait for the verification.
